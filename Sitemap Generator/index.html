Sitemap Generator

For IDL Website

Sitemap: a file containing a list of URLs for search engines to crawl

The Industry Documents Library webiste has lacked a sitemap for the past few years, ever since {****}. Creating the sitemap for IDL is a difficult task because their database of documents includes some 14.8 million documents, each with their own unique URL that must be listed in the sitemap. Additionally, we needed not only a sitemap for all the current URLs, but rather a programatic solution that regenerates a sitemap every time the site is updated, keeping search engines up to date on what sites need to be crawled.
